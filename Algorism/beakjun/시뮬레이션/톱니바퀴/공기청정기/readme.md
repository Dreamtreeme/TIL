🌬️ 백준 17144 "미세먼지 안녕!" 풀이 흐름 (자연어 설명)

1. 입력 단계

격자의 크기 **R(행), C(열), T(시간)**을 입력받는다.

R×C 크기의 보드 상태를 입력받는다.

각 칸에는 미세먼지 양(0 이상) 또는 -1(공기청정기)이 들어있다.

공기청정기는 항상 세로로 두 칸 연속해서 배치되어 있으므로, 두 위치를 따로 저장한다.

2. 시뮬레이션 반복 (T번)

매 초마다 두 단계를 거친다:

(1) 확산 단계

격자의 모든 칸을 순차적으로 보면서,
현재 칸의 미세먼지가 있으면 상하좌우로 퍼뜨릴 양을 계산한다.

확산되는 양 = (해당 칸의 먼지 양) // 5

확산은 인접 칸이 범위를 벗어나지 않고, 공기청정기가 아닌 경우에만 가능하다.

확산된 먼지는 임시 보드(temp)에 저장한다.
→ 이렇게 하면 "모든 칸에서 동시에 확산"된 효과를 낼 수 있다.

최종적으로, temp의 값을 원래 보드에 합쳐서 갱신한다.

(2) 공기청정기 순환 단계

공기청정기는 두 부분으로 나뉜다:

윗부분 : 공기가 반시계 방향으로 순환한다.

아랫부분 : 공기가 시계 방향으로 순환한다.

바람이 순환하면서 먼지가 한 칸씩 이동한다.

공기청정기 바로 옆 칸은 항상 0이 된다 (깨끗한 공기 유입).

3. 종료 후 계산

T초가 모두 지나면,
보드에 남아있는 모든 미세먼지의 양을 합산한다.

공기청정기 위치(-1)는 합산에서 제외한다.

이 합산값을 출력한다.
