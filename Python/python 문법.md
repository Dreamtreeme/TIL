number = 10
double = 2* number
print(double)

number = 5
print(double)

## number가 뒤에 바뀌어도 double의 메모리 주소는 바뀌지 않는다
## 따라서 number의 첫째주소 *2를 한 값이 double에 할당되고 그 할당값은 다시 바뀌지 않는다.
